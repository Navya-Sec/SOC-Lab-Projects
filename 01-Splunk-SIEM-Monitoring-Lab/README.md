# AI-Augmented SOC Lab Using Splunk, Wazuh & Claude AI

## Overview

This project demonstrates the implementation of an AI-Augmented Security Operations Center (SOC) lab designed to simulate a real-world enterprise SOC environment. The lab integrates Splunk Enterprise, Sysmon, Wazuh, and Claude AI to perform centralized log collection, security monitoring, detection engineering, attack simulation, and AI-assisted incident investigation.

## Objectives

* Deploy Splunk Enterprise Free Edition for centralized log management.
* Collect Windows Security Logs and Sysmon logs.
* Integrate Wazuh alerts into Splunk.
* Develop dashboards for security monitoring.
* Create SPL detection rules for suspicious activities.
* Integrate Claude AI using Model Context Protocol (MCP) for AI-assisted investigations.
* Simulate attacks and validate detections.
