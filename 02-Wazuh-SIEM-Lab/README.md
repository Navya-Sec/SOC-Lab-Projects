# Wazuh SIEM Installation & Configuration Lab

## Overview

This project demonstrates the installation and initial configuration of Wazuh SIEM in a virtual lab environment. The objective was to deploy Wazuh from scratch, configure its core components, verify successful installation, and document each step with commands, screenshots, issues encountered, and troubleshooting methods.

## Objectives

- Install Wazuh SIEM from scratch.
- Configure the Wazuh Manager, Indexer, and Dashboard.
- Verify communication between Wazuh components.
- Access the Wazuh web dashboard.
- Document the complete installation process.
- Record installation issues and troubleshooting steps.

## Lab Environment

- Operating System: Ubuntu Server
- Virtualization Platform: VMware Workstation / VirtualBox
- Wazuh Version: 4.x
- Browser: Google Chrome

## Installation Process

The installation included:

- Preparing the Ubuntu server
- Updating system packages
- Installing Wazuh components
- Configuring services
- Starting and verifying Wazuh services
- Accessing the Wazuh Dashboard
- Validating successful deployment

## Documentation

The complete installation report includes:

- Installation steps
- Commands used
- Screenshots
- Issues encountered
- Troubleshooting
- Final verification
